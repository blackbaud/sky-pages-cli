build           Builds the current SPA into the dist/ folder.

e2e             Runs any end-to-end tests in the e2e folder.
help            Displays help for SKY UX CLI arguments.
lint            Validates Typescript code against rules.
new             Initializes a new local SKY UX application.
serve           Serves the current application locally.
test            Runs any unit tests in the src/app folder.

watch           Runs unit tests and watches the file system for changes.
